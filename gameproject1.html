SUDDEN

Unreal Engine Blender Blueprint

Project Overview

This is a first-person survival-horror game where the player is trapped inside a dark, hostile environment while being hunted by a terrifying monster. The primary goal of the game is to locate the monster’s own weapon, a special item that spawns randomly in different areas of the map. This weapon is the only way to kill the monster and escape the environment. Until the player finds it, they must rely on stealth, exploration, and awareness to stay alive.

My Contributions

  • Designed all core gameplay systems including player movement, interaction, survival mechanics, and enemy logic.
  • Created the complete level design, lighting, environment setup, and horror atmosphere.

Technical Implementation

AI Systems

  • Built the monster’s complete AI behavior using Blueprints.
  • Implemented patrol, chase, search, and attack states for realistic enemy movement.
  • Used NavMesh-based pathfinding for smooth navigation throughout the level.
  • Created random patrol logic to make monster behavior unpredictable.
  • Designed the AI to respond dynamically to player actions and environment.

Weapon Framework

  • Implemented a random weapon spawn system, placing the monster’s weapon at different locations every playthrough.
  • Built the pickup system, allowing the player to collect and equip the weapon.
  • Created the logic where only the monster’s own weapon can kill it, adding unique gameplay challenge.
  • Added UI prompts and interaction feedback for weapon pickup.
  • Integrated sound and visual effects when the weapon is found or used.
  • Developed the final kill mechanic, ending the game once the monster is defeated.

Performance

  • Optimized all Blueprints to reduce unnecessary logic and improve execution speed.
  • Improved AI performance by minimizing Tick usage and using events where possible.
  • Reduced heavy operations like traces and perception checks for smoother gameplay.
  • Cleaned unused assets, optimized collision, and reduced texture sizes where needed.
  • Performed multiple tests to ensure stable performance and responsive gameplay.
