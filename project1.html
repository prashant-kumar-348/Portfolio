Project Overview

This project is a cinematic sci-fi hangar sequence created inside Unreal Engine 5, featuring a large-scale spaceship, troops, and a dramatic space-view environment. The goal of this project was to practice environment design, lighting, camera work, and cinematic storytelling using real-time rendering.

My Contributions

    Environment Design

  • Created the entire hangar environment inside UE5.
  • Composed the scene using free modular sci-fi assets.
  • Designed the layout to support large-scale troop formations and a central landing platform.

    Lighting & Atmosphere

  • Set up Lumen-based real-time lighting.
  • Added soft volumetric fog and environmental bounce light for realism.
  • Used emissive sci-fi lights and reflective materials to enhance the futuristic style.

    Camera Cinematics

  • Created a full cinematic shot using Sequencer.
  • Set up smooth camera movements for dramatic reveals.
  • Adjusted depth of field, exposure, and keyframes for a film-like experience.

    Animation & Character Setup

  • Used free character animations and integrated them into the soldiers.
  • Synced idle and formation animations to enhance troop realism.
  • Ensured proper animation looping and playback inside the cinematic timeline.

    Technical Work

  • Optimized scene performance.
  • Organized assets in UE5 following a clean folder structure.
  • Balanced lighting and shadows to avoid overexposure in real-time rendering.

Tools & Technologies

  • Unreal Engine 5 (Lumen, Sequencer, Niagara).
  • Free marketplace sci-fi models & animations.
  • Real-time rendering tools.
  • Level design, lighting, and cinematic tools in UE5.
